Welcome to Digital Clock

I have created a code to show Digital Clock into your website. I have created this code using HTML and Jquery. You can use this code show Digital Clock into any website you want.

HTML & Jquery

To use my code of Digital Clock you must have a basic knowledge of HTML & jQuery. As i mentioned earlier that this code is created using HTML & jQuery. You will need the jQuery library to run this code. You can download the latest jQuery from Here I am using jQuery 1.10.2

Once you get the jQuery, you will need to copy the jQuery file to somewhere into your project. The code to add the jQuery to your HTML page is:

<script type="text/javascript" src="jquery-1.10.2.js"></script>

Note: you can change your path of "src" to something like "js/jquery-1.10.2.js".

So, we are ready to go with HTML code now First you have to copy the below code and paste it somewhere into your HTML page where ever you want to show the Digital Clock.

Code Goes like this

HTML Code: <body onload="startTime()"> <div> <div id="time"></div> </div> </body>

jQuery Code:

`<script>
    function startTime() {
        var today = new Date();
        var h = today.getHours();
        var m = today.getMinutes();
        var s = today.getSeconds();
        // add a zero in front of numbers<10
        m = checkTime(m);
        s = checkTime(s);
        document.getElementById('time').innerHTML = h + ":" + m + ":" + s;
        t = setTimeout(function () { startTime() }, 500);
    }

    function checkTime(i) {
        if (i < 10) {
            i = "0" + i;
        }
        return i;
    }
</script>`

How it works

In the "BODY" tag i have called a function "startTime()" on onload. It means when body loads it will call the function to start working and show time. There should some element like "Div" to have a unique ID. This "Div" will be fetched by the jQuery function "startTime()" to show the current time into that "Div".

jQuery funtion "startTime()" will calculate the time using "Date()" function of jQuery. Then it will save the hour, minutes and seconds in some variables. Now, the "Div" with "ID" as "time" is fetched and all the variables are displayed into it. "checkTime()" function is created to add zero in from of single digits.
